## Overview

BeatForge is a professional digital audio workstation (DAW) that runs entirely in your browser. Instantly experience a full-featured music production environment—no install required.

**Features:**
- Real DAW interface inspired by Ableton, FL Studio, and Logic
- Interactive product demo mode (guided walkthrough)
- Preloaded demo project (Kick, Snare, HiHat, Bass, Lead Synth)
- Play, move clips, adjust BPM, trigger sequencer, mix tracks
- Modern dark UI, premium design, and fast performance
- Built with React, Vite, Web Audio API, Framer Motion

## Architecture

```
/beatforge
	/app/full-daw-interface      # Full DAW UI (App Mode)
	/demo/guided-product-tour    # Interactive Demo Mode
	/components                  # UI component library
	/audio-engine                # Audio engine modules
	/ui-system                   # Shared UI system
	/docs                        # Documentation, screenshots
	index.html                   # Landing page
```

## User Experience

1. **Landing Page:**
	- Hero section, product branding, and navigation
	- [Launch App] (full DAW) | [Watch Demo] (guided tour) | [View Source]
2. **App Mode:**
	- Full interactive DAW UI, demo project loaded
	- Play demo beat, move clips, adjust controls
3. **Demo Mode:**
	- Step-by-step product walkthrough with UI highlights and animations
	- See sequencer, piano roll, mixer, and collaboration in action
